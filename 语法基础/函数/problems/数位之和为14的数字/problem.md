# 数位之和为14的数字

## 题目描述

**小谭** 克服了对 $1$ 和 $4$ 两个数字的恐惧，现在他很喜欢这两个数字。
现在他想输入两个正整数 $n$ 和 $m$，输出它们之间所有数位之和为 $14$ 的数字。

注意：请使用自定义函数来实现此功能。

## 输入格式

输入两个正整数 $n$ 和 $m$。

$1 \leq n \lt m \leq 10^6$。

## 输出格式

分行输出所有数位之和为 $14$ 的数字。

特别的，如果这个范围内没有满足条件的数字，你需要输出一个 $-1$。

## 输入输出样例

### 输入 #1

```
800 1000
```

### 输出 #1

```
806
815
824
833
842
851
860
905
914
923
932
941
950
```
