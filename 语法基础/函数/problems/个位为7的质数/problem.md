# 个位为7的质数

## 题目描述

输入两个正整数 $n$ 和 $m$，输出 $n - m$ 中所有个位为 $7$ 的质数，若范围中没有则输出 `-1`。

注意：请使用自定义函数来实现判断质数的功能。

## 输入格式

输入两个正整数 $n$ 和 $m$。

$1 \leq n \lt m \leq 10^6$。

## 输出格式

分行输出范围内的所有个位为 $7$ 的质数，若没有，则输出 `-1`。

## 输入输出样例

### 输入 #1

```
2
20
```

### 输出 #1

```
7
17
```

### 输入 #2

```
2
5
```

### 输出 #2

```
-1
```
